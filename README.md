## Install COSMIC on Arch Linux

![COSMIC screenshot](https://i.e33.io/img0/Screenshot_2026-02-25_20-15-34.jpg)

### Features:
- Base set of apps for a ready-to-use desktop session

### Notes:
- Custom theming configs are WIP

### Installation:
```bash
# clone repo
git clone https://github.com/e33io/arch-cosmic

# change directory
cd arch-cosmic

# run script
sh install-cosmic.sh
```
