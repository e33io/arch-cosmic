## Install COSMIC (desktop environment) on Arch Linux

![COSMIC screenshot](https://i.e33.io/img0/screenshot_2025-10-17_07-43-34.jpg)

### Features:
- Base set of apps for a ready-to-use desktop session

### Installation:
```bash
# clone repo
git clone https://github.com/e33io/arch-cosmic

# change directory
cd arch-cosmic

# run script
sh install-cosmic.sh
```
